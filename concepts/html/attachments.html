Linking is better than embedding in all respects, except that 1) the result of the link can change thus ruining any immutability or reliability of a document using links, and 2) the result of the link can be unavailable since it's not included embedded inside the document. For 1) use CHKs to link, so that the result can never change. For 2) you can append pieces after the document to have the same reliability as embedding, with more efficiency/space. Plus using CHKs lets everyone link to the same file, and each only download it once, unlike w/ email.
